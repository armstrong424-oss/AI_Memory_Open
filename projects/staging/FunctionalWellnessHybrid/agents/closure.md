# closure.md — FunctionalWellnessHybrid Staging Session Closure

Run at the end of any session.

1. **Finalize the decision log** in `decisions-learnings/` (decisions, learnings, blockers, keywords).
2. **Update `Key-Decisions.md`** — add a row for this session's log.
3. **Update `Sessions.md`** — add a session row.
4. **Update `NextSteps.md`** — progress, status fields, validation to-dos, blockers.
5. **Update `Master-Context.md`** — document Status fields and any refined content; bump Last Updated.
6. **Confirm saved**, then remind the owner to `git add . && git commit && git push` (Cowork sandbox cannot push).
7. **Report**: what changed, what's blocked, and the top next step.
