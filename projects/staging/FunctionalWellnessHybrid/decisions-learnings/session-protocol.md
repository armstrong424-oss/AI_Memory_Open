# Decision & Learnings Log Protocol — FunctionalWellnessHybrid

Each session, create a timestamped decision log at:
`decisions-learnings/Key-Decisions-[YYYY-MM-DD]_[HHMM].md`

## Template

```markdown
# Key Decisions — {YYYY-MM-DD} {HHMM}

**Session Goal:** {what we aimed to accomplish}

## Decisions Made
### Decision 1: {Title}
- **Context:** Why did we face this decision?
- **Options Considered:** Alternatives?
- **Decision:** What did we choose?
- **Rationale:** Why?
- **Impact:** What does it affect?

## Learnings & Insights
- {key insight}

## Blockers / Open Questions
- {unresolved issue}

## Keywords (for indexing)
- keyword1, keyword2
```

## How to Index
After finalizing, add a row to `Key-Decisions.md`:

| Date | Keywords | File | Summary |
|---|---|---|---|
| YYYY-MM-DD | k1, k2 | `Key-Decisions-YYYY-MM-DD_HHMM.md` | one-sentence summary |
